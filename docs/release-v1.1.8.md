# TongYang Customs Platform v1.1.8

AI Semantic Customs Document Understanding Engine.

- Strengthens DS2 / Taiwan import declaration recognition from customs fields, table structure, and partial OCR evidence.
- Prevents uploaded declaration candidates from being shown as missing documents when confidence is low.
- Adds dynamic evidence columns for INV, PL, B/L, Arrival Notice, SO, Booking, DS2 declaration, tax sheet, clearance list, and drawback standard.
- Replaces fixed INV / PACKING / B/L / declaration compare columns with a case-driven evidence table.
- Removes developer wording from normal ERP messages and keeps tracebacks in runtime logs.
- Improves table contrast, selected-state colors, high-DPI scaling, and minimum window sizing.
